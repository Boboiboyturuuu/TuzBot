## TERMUX USER
```bash
$ pkg upgrade && pkg update
$ pkg install git -y
$ pkg install nodejs -y
$ pkg install ffmpeg -y
$ pkg install imagemagick -y
$ git clone https://github.com/Boboiboyturuuu/TuzBot/
$ cd TuzBot-master
$ npm i 
```

## OPTIONS
```bash
$ node . --qr
$ node . --mobile
$ node . --pairing-code
$ node . --singleauth
```

#### If npm install failed, try using yarn instead of npm
```bash
$ pkg install yarn -y
$ yarn install
```
